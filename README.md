# Customer Segmentation using K-Means Clustering

## Project Overview
This project performs **unsupervised learning** on the Mall Customer dataset to segment customers based on **Annual Income** and **Spending Score**. 

## Features
- Exploratory Data Analysis (EDA)
- K-Means clustering
- Cluster visualization
- Cluster interpretation for marketing insights

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- Scikit-Learn (K-Means)

## Dataset
[Mall Customers Dataset](https://www.kaggle.com/datasets/shwetabh123/mall-customers)
